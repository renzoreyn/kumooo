# Contributing

```bash
pnpm install
pnpm --filter @kumooo/cli build
pnpm ci
```

## Ground rules

- TypeScript strict. Escaping by default. Tenancy on every query.
- Migrations are additive only.
- Writing follows [WRITING.md](WRITING.md). Em dashes are banned.
- One change per PR. Explain the why.

## Security

Email [contact@renzoreyn.dev](mailto:contact@renzoreyn.dev). Don't open a public issue for security problems.
