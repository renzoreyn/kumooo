# Your first kumooo.js site

This is the fun part. One command scaffolds a Next.js app with the kumooo.js UI kit.

## Create

In a terminal, go somewhere you keep projects, then:

```bash
npx create-kumooo
```

Pick a folder name. Pick **blank**, **blog**, or **shop**. Blank is fine for learning.

## Install and run

```bash
cd your-folder-name
pnpm install
pnpm dev
```

If you use npm instead:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). That is your site, only on your machine.

## What you are looking at

A starter page using `@kumooo/ui`. Change text in `app/page.tsx`, save, and the browser updates. That loop is most of web development.

## Stuck?

- `npx` not found → Node is not installed or the terminal is old. See [Set up your computer](/learn/setup-your-computer).
- Port in use → something else is on 3000. Stop it, or start Next on another port.
- Prefer one-click hosting first → [Deploy on Vercel from the homepage](/) or use [Guided setup](/setup).
