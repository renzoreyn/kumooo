## You need three things

1. **Node.js LTS** from [nodejs.org](https://nodejs.org). Check with `node -v`.
2. A **code editor**. Cursor or VS Code. We do not care which.
3. About **ten minutes** and a terminal.

## Create a project

```bash
npx create-kumooo
```

Pick blank, blog, or shop. You get a working Next.js app in a new folder.

## Run it locally

```bash
cd my-site
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000). Edit `app/page.tsx`. Watch it update.

## Deploy

Push to GitHub and connect the repo to Cloudflare, or use kumooo hosting at `{slug}.kumooo.site`. [Guided setup](/docs/setup) has the clicks.

See [Hosting](/docs/hosting) and [Pricing](https://kumooo.dev/pricing) for Nimbus quotas and the rest of the cloud names.
