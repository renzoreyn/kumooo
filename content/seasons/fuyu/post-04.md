# cold start checklist

Before you pick fuyu:

- [ ] titles are labels, not headlines
- [ ] posts under 400 words feel normal
- [ ] you like dark UI
- [ ] readers expect density

If two boxes unchecked, try haru.

If all checked, publish logs here.

Example line format:

`YYYY-MM-DD / topic`

Body: bullets, code, facts. No flourish.

Anti-patterns for this theme:

- long metaphor openings
- hero images above fold
- marketing adjectives without numbers

Good patterns:

- semver in title
- timestamps in UTC when incidents matter
- links to PRs or commits

Link platform: https://kumooo.dev

Dashboard: https://dash.kumooo.dev

Sample title formats that work:

- `2026-02-01 / deploy`
- `STATUS: api-v1`
- `incident-2026-02-03-dns`

Sample title formats that fight the theme:

- `Ten thoughts on creativity`
- `Why we are excited to announce`

Match format to theme. Mismatch looks like wearing a coat indoors.

Word count target for fuyu demos: 150-400. Still log-shaped. Still scannable. Prose optional.
