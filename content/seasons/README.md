# Season theme demo content

Markdown bodies for the four official Kumooo season showcase sites: `haru`, `natsu`, `aki`, and `fuyu`.

Each folder holds one `about.md` page and four sample posts (`post-01.md` through `post-04.md`). No frontmatter; titles are the first `#` heading.

These files are seeded into the official org by `scripts/seed-official.mjs` when you run it against a local or deployed API.
