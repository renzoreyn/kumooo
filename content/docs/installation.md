# Installation

## Create an app

```bash
npx create-kumooo my-app
cd my-app
pnpm install
pnpm dev
```

## Monorepo (contributors)

```bash
pnpm install
pnpm dev
```

## Requirements

| Thing | Version |
|---|---|
| Node.js | 20+ |
| pnpm | 9+ |

No VPS required locally. Deploy anywhere Next.js runs (Vercel, Node host, etc.).
