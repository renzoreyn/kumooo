# Media and branding

Images live in R2. Branding is two fields on the site: logo and favicon.

## Upload media

1. Open your site in the dashboard.
2. Go to **Media**.
3. Upload. Each site has a **150 MB** media quota.

Use Media for logos, favicons, post images, and anything else you link from Markdown.

## Logo

**Design / Branding** → pick a media file for the logo.

- Themes that support branding show the image in the header.
- If you leave it empty, the site title text is the brand.

## Favicon

Same page. Upload a square PNG or SVG.

If you skip it, Kumooo serves the default geometric **k** mark (mint dot) so the tab is never blank.

## Quotas

Uploads that would exceed 150 MB are rejected. Delete unused files in Media when you need room.

## Markdown images

Reference uploaded media by public URL, or paste the path your theme expects. Prefer relative or same-origin URLs so cache and CSP stay simple.
